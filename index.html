What's New with INPOS?

This is the INPOS first release! We are happy that you are one of the first people to see our OS!

Features

There is a web browser in INPOS. To get the web browser, go to the top bar -> Open -> Web Browser. You can type a website that starts with HTTPS://, HTTP://, or simply type something to perform a Google search! Currently, there is no built-in INPOS browser, so it will open your default browser on your Windows computer.

Future Updates

We plan to add the INPOS browser to INPOS in the future. The browser already has a name: Inbrowser. There will be a pro version of INPOS (Owner Approved), INPOS Pro will have an exclusive wallpaper in it.

Please note that these features are not 100% guaranteed to be included.
